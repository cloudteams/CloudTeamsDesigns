# CloudTeams

## Development

1. Add file package.user.json

	```json
	{
		"sync-dir": "path/to/sync_folder/",
		"OS"      : "mac|windows"
	}
	```

2. Install node modules

	`npm install`

3. (a) Working with project 

	`npm run watch`

3. (b) For working with a browser sync / livereload

	`npm run watch-live`
